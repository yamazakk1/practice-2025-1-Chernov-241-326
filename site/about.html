О проекте

Киберполигон — это образовательная платформа, направленная на обучение кибербезопасности и веб-разработке. Проект создан для студентов, которые хотят освоить современные технологии и получить практический опыт.

CTF-задания

Практические задачи по кибербезопасности в формате Capture The Flag.

Веб-разработка

Изучение Django, HTML, CSS и других технологий для создания веб-приложений.

Практика

Реальные проекты для портфолио и профессионального роста.

Для кого это?

  • Студенты IT-направлений
  • Начинающие разработчики
  • Преподаватели и наставники

Технологии проекта

  • Backend: Django, SQLite
  • Frontend: HTML, CSS
  • Инфраструктура: Vagrant, VirtualBox
