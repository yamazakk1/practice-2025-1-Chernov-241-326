Pastebin-сервер

В рамках проектной практики я разработал pastebin-сервер — веб-приложение для обмена текстовыми данными. Сервер позволяет пользователям создавать текстовые пасты, делиться ими через уникальные ссылки и автоматически удаляет устаревшие записи. Это вариативное задание, выполненное на основе идей из codecrafters-io/build-your-own-x.

Создание паст

Пользователи могут отправить текст через форму и выбрать срок действия: 1 час, 1 день или 1 неделя.

Просмотр паст

Каждая паста доступна по уникальному URL вида /paste/<slug>, с безопасным отображением текста.

Автоматическая очистка

Устаревшие пасты удаляются каждые 30 секунд с помощью фоновой задачи.

Консольное управление

Команды stop и count позволяют остановить сервер или узнать количество паст.

Технологии

  • Backend: Go (net/http), PostgreSQL (pgx/v5)
  • Инструменты: Git, Markdown, PlantUML

Как это работает?

Сервер обрабатывает HTTP-запросы (GET и POST), сохраняет пасты в PostgreSQL и использует горутины для периодической проверки сроков действия. Пользовательский интерфейс — минималистичный, с формой для создания паст и страницами для их просмотра. Консольные команды упрощают управление сервером.

На главной странице доступна форма для создания пасты с выбором срока действия.

Паста отображается по уникальной ссылке, с экранированием HTML для безопасности.

Архитектура сервера: обработка запросов, взаимодействие с PostgreSQL и фоновая очистка.
